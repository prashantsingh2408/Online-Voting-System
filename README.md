# Online voting system

## Admin Features
* Login
* Manage Parties
* Manage Users
* Check Poll Results
* Update Password
* Update Profile
* Block & unblock users
* Restart Voting
* Announce the result.
* Start and stop voting using timer (Read rule no 7)
* Logout
## Voter/User Features
* Registration
* Login
* Update Profile
* Email verification using OTP or token
* Give Vote
* Forget password
